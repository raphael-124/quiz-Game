# Quiz Game — IQ Practice

> Train your reasoning with hundreds of IQ-style questions — logic, patterns, verbal reasoning, number series, and memory challenges.

## Description
`Quiz Game` is a lightweight Python quiz app that helps you practice IQ-style questions. It includes practice and timed modes, step-by-step explanations, and progress tracking. Ideal for students, job-test preparation, and anyone who wants daily brain training.

## Features
- Hundreds of IQ-style questions across categories (Logic, Verbal, Numeric, Memory)
- Practice mode (untimed) and Timed mode (simulate test conditions)
- Explanations for answers so you learn as you play
- Progress tracking with local history
- Simple command-line interface — no account required

## Quick Start
1. Make sure you have Python 3.8+ installed.
2. From the project root, run:

```bash
python main.py
```

The app will present category choices and start the quiz in the chosen mode.

## Files
- `main.py` — application entry point
- `data.py` — question bank and data loader
- `question_model.py` — question model and utilities
